12誘導擬似心電図生成プログラム

擬似的な12誘導心電図波形を生成するC++プログラムである。観測された12誘導心電図波形のCSV形式ファイルを引数に与えると、数理モデルを自動構成し過不足のないパラメータの一覧を返すプログラムと、パラメータを与えると擬似的な12誘導心電図波形のCSV形式ファイルを出力するプログラムからなる。C++言語がコンパイルできる環境。およびlibconfigライブラリが使える環境（典型的にはLinux）で実行可能である。配布条件はMITライセンスである。産総研知財管理番号2025PRO-3230に基づいて開示される。

本プログラムについての連絡先は shiro@aist.go.jp である。
